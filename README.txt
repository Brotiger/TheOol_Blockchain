1)Необходимо установить:

git - система контроля версий
docker - система виртуализации и контейнеризации
docker-compose (под windows возможно идет вместе с docker)- система управления несколькими контейнерами сразу
python 3.X (чем новее тем лучше) - язык программирования
pip (или pip3, под windows возможно идет вместе с python) - пакетный менеджер python

2)Для проверки установки:
Необходимо вбить следующие команды в терминал (командную строку)

Вызов командной строки: windows + R и вводим cmd в окно "Выполнить". Затем нажимаем Ок
(Если ПО установленно то данные команды ниже должны вывести версию ПО по каждому пункту)

git --version
docker version
python -V
pip -V (или pip3 -V)

3)Переносим проект на локальную машину
Открываем командную строку и переходим в домашнюю папку пользователя:

cd ~
копируем код проекта с bibucket:

git clone https://Brotiger@bitbucket.org/smile1601/theoolias.git

4) Далее необходимо задать переменные окружения:
Необходимо вбить следующие команды в терминал (командную строку)

SET VOLUME_DIR=Путь до директории где лежит данный файл (например SET VOLUME_DIR=С:\Users\Brotiger\theoolias)
SET MYSQL_ROOT_PASSWORD=password

5)Запускаем сам докер - иконку докера должна быть где то на рабочем столе

6)Запускаем контейнеры
Заходим в командную строку и переходим в директорию с проектом
cd ~/theoolias

Выполняем команду для старта контейнеров:

docker-compose up
